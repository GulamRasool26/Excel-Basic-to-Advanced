# Advanced Excel Practice — Data Analytics Techniques

I already use Excel regularly for data cleaning and dashboarding (see my 
[Biker Buyers Dashboard] and [Survey Analysis] projects). This repo is 
where I'm deliberately reinforcing and extending specific advanced 
techniques that come up frequently in Data Analyst job requirements — 
Power Query, calculated-field pivot tables, XLOOKUP/INDEX-MATCH, and 
basic DAX — to make sure they're second nature, not just familiar.

## Focus areas
- Power Query: data cleaning and transformation without manual edits
- Pivot Tables with calculated fields (beyond basic grouping)
- XLOOKUP / INDEX-MATCH for multi-table lookups
- Basic DAX measures (SUMX, CALCULATE) via Power Pivot
- A small end-to-end dashboard combining the above

## Why this matters
Excel and SQL are the two most frequently required skills across 43 real 
Pakistan data analyst postings I analyzed in [this project] — this repo 
is targeted reinforcement of the Excel side specifically.
